# 📝 08 – Survey System

🇨🇳 简易问卷系统：前端本地处理与统计结果显示，无需后端。  
🇩🇪 Einfaches Umfragesystem: lokale Datenspeicherung & Auswertung ohne Backend.  
🇬🇧 Simple survey app with client-side data handling and instant results.

**Key Concepts / 核心概念 / Zentrale Konzepte**
- 表单收集与 `FormData` 处理  
- `localStorage` 数据存储  
- 使用 `<canvas>` 绘制简易柱状图  
- 动态生成统计汇总与结果展示
